love.forte.simbot.common.api.message.containers 0.0.1 API

See: Description

  • Interface Summary 
    Interface Description
    OriginalDataContainer原始数据容器。 定义可以得到原始数据的字符串信息。
    AccountCodeContainer账户号码容器。定义可以得到一个账号的号码 其中,codeNumber默认为code.
    AccountNicknameContainer账户昵称容器。定义可以得到一个账号的昵称 昵称可能不存在。
    AccountRemarkContainer账户备注容器。定义可以得到一个账号的备注信息。可能是好友备注或者群名片。 备注可能不存在
    AccountNameContainer账户名称容器，即 账号昵称容器 与 账号备注容器 并提供几个默认的整合值:
    AccountAvatarContainer账户头像容器，定义可以得到一个头像链接。 头像不是必须的，可能会不存在。
    AccountContainer一个账号信息容器, 继承了 用户名称容器用户头像容器用户账号容器
    GroupCodeContainer群号容器。定义可以得到一个群的群号
    GroupAvatarContainer群头像容器。定义可以得到群头像 头像不是必须的，可能会不存在
    GroupNameContainer群名称容器，定义可以得到群的群名称
    GroupContainer群信息容器，实现 群头像容器群账号容器群名称容器
    BotCodeContainer机器人自身的账号容器
    BotNameContainer机器人自身的名称容器
    BotAvatarContainer机器人自身的头像容器 头像不是必须的，可能不存在
    BotContainer机器人基础信息容器, 其实现了 机器人账号容器机器人名称容器机器人头像容器
    FlagContainer标识容器。定义可以得到一个字符串标识。 但是一般来讲, MsgGet的FlagContainer实现容器基本上都可以通过MsgGet.id来实现
