    Interface Description
    AccountCodeContainer账户号码容器。定义可以得到一个账号的号码
    AccountNicknameContainer账户昵称容器。定义可以得到一个账号的昵称
    AccountRemarkContainer账户备注容器。定义可以得到一个账号的备注信息。可能是好友备注或者群名片。 备注可能不存在
    AccountNameContainer账户名称容器，即 账号昵称容器 与 账号备注容器 并提供几个默认的整合值:
    AccountAvatarContainer账户头像容器，定义可以得到一个头像链接。 头像不是必须的，可能会不存在。
    AccountInfoContainer一个账号信息容器, 继承了 用户名称容器, 用户头像容器, 用户账号容器
    AccountContainer账号容器，可以得到一个账号的信息
    BotCodeContainer机器人自身的账号容器
    BotNameContainer机器人自身的名称容器
    BotAvatarContainer机器人自身的头像容器 头像不是必须的，可能不存在
    BotInfoContainer机器人基础信息容器, 其实现了 机器人账号容器, 机器人名称容器, 机器人头像容器
    BotContainerbot容器，可以得到一个 bot信息
    OriginalDataContainer原始数据容器。 定义可以得到原始数据的字符串信息。
    PermissionContainer权限容器，定义可以得到一个 权限。一般代表这个人在群里的权限
    FlagContainer标识容器。定义可以得到一个标识。
    ActionMotivationContainer行动动机容器, 定义可以得到当前类型的动机类型。一般来讲，此容器使用在枚举类上，例如消息事件中特有的类型枚举.
    GroupCodeContainer群号容器。定义可以得到一个群的群号
    GroupAvatarContainer群头像容器。定义可以得到群头像 头像不是必须的，可能会不存在
    GroupNameContainer群名称容器，定义可以得到群的群名称
    GroupInfoContainer群信息容器，实现 群头像容器, 群账号容器, 群名称容器
    GroupContainer可以得到一个群信息容器
    OperatorCodeContainer操作者code容器
    BeOperatorCodeContainer被操作者code容器
    OperatorNicknameContainer操作者昵称容器，定义可以得到操作者的昵称信息.
    BeOperatorNicknameContainer被操作者昵称容器，定义可以得到被操作者的昵称信息.
    OperatorRemarkContainer操作者备注容器，定义可以得到操作者的备注信息.
    BeOperatorRemarkContainer被操作者备注容器，定义可以得到被操作者的备注信息.
    OperatorNameContainer操作者名称容器，定义可以得到操作者的昵称与备注信息.
    BeOperatorNameContainer被操作者名称容器，定义可以得到被操作者的昵称与备注信息.
    OperatorAvatarContainer操作者头像容器，定义可以得到一个头像链接。 头像不是必须的，可能会不存在。
    BeOperatorAvatarContainer被操作者头像容器，定义可以得到一个头像链接。 头像不是必须的，可能会不存在。
    OperatorInfoContainer操作者信息容器
    OperatorContainer可以得到操作者信息的容器
    BeOperatorInfoContainer被操作者信息容器
    BeOperatorContainer可以得到被操作者信息的容器
    OperatingContainer针对一个存在 操作 内容的事件, 此容器提供了在 操作 事件中的 操作者 与 被操作者 的相关信息容器
